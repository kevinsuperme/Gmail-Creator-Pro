# 02 - 环境配置与安装

本文档详细介绍在 Windows、Linux、Termux（Android）三大平台的安装部署步骤。

## 一、系统要求

### 1.1 硬件要求

| 项目 | 最低配置 | 推荐配置 |
|------|---------|---------|
| 内存 (RAM) | 4 GB | 8 GB |
| 磁盘空间 | 500 MB | 1 GB |
| 网络 | 稳定连接 | 稳定连接 + 代理 |

### 1.2 软件要求

| 软件 | 版本要求 | 说明 |
|------|---------|------|
| Python | 3.8+（推荐 3.12） | 核心运行环境 |
| Google Chrome | 最新版 | Selenium 驱动的目标浏览器 |
| Git | 任意版本 | 用于克隆仓库 |
| pip | 与 Python 配套 | 安装依赖包 |

> ⚠️ **关键**：Chrome 浏览器版本必须与 ChromeDriver 匹配。项目使用 `webdriver-manager` 自动管理，但本地 Chrome 必须为最新版。

---

## 二、Windows 平台安装

### 2.1 安装基础软件

#### 步骤 1：安装 Python

1. 访问 [python.org](https://www.python.org/downloads/) 下载 Python 3.12
2. 运行安装程序，**务必勾选 "Add Python to PATH"**
3. 验证安装：
   ```powershell
   python --version
   pip --version
   ```

#### 步骤 2：安装 Google Chrome

1. 访问 [google.com/chrome](https://www.google.com/chrome/) 下载安装
2. 保持自动更新开启

#### 步骤 3：安装 Git

1. 访问 [git-scm.com](https://git-scm.com/download/win) 下载安装
2. 验证安装：
   ```powershell
   git --version
   ```

### 2.2 获取项目代码

```powershell
# 方式一：克隆仓库
git clone https://github.com/ShadowHackrs/gmail-account-creator.git
cd gmail-account-creator

# 方式二：直接下载已存在的项目
# 若已有项目目录，直接进入即可
cd e:\gmail-account-creator
```

### 2.3 安装 Python 依赖

```powershell
# 推荐先创建虚拟环境
python -m venv venv
.\venv\Scripts\Activate.ps1

# 安装所有依赖
pip install -r requirements.txt
```

**依赖清单说明：**

| 包名 | 版本 | 用途 |
|------|------|------|
| selenium | >=4.15.0 | 浏览器自动化 |
| webdriver-manager | >=4.0.0 | 自动下载 ChromeDriver |
| rich | >=13.7.0 | 终端美化输出 |
| requests | >=2.31.0 | HTTP 请求 |
| unidecode | >=1.3.7 | 文本规范化 |
| beautifulsoup4 | >=4.12.0 | HTML 解析 |
| fp | >=0.1.0 | 免费代理获取 |

### 2.4 验证安装

```powershell
python -c "import selenium; import rich; import requests; print('依赖安装成功')"
```

若输出 "依赖安装成功" 则环境配置完成。

## 三、Linux 平台安装（Kali / Ubuntu / Debian）

### 3.1 安装基础软件

```bash
# 更新系统包
sudo apt update && sudo apt upgrade -y

# 安装 Python、pip、git
sudo apt install python3 python3-pip git -y

# 安装 Chrome（以 Kali 为例）
wget https://dl.google.com/linux/direct/google-chrome-stable_current_amd64.deb
sudo apt install ./google-chrome-stable_current_amd64.deb -y
```

### 3.2 获取并运行项目

```bash
# 克隆仓库
git clone https://github.com/ShadowHackrs/gmail-account-creator.git
cd gmail-account-creator

# 安装依赖
pip3 install -r requirements.txt

# 运行
python3 auto_gmail_creator.py
```

## 四、Termux 平台安装（Android）

根据官方指南，在 Android 设备上通过 Termux 部署的步骤如下：

### 4.1 安装 Termux

从 [F-Droid](https://f-droid.org/packages/com.termux/) 下载安装（Google Play 版本已过时，不推荐）。

### 4.2 安装步骤

```bash
# 1. 更新系统
pkg update && pkg upgrade -y

# 2. 安装 Python 和 Git
pkg install python git -y

# 3. 克隆仓库
git clone https://github.com/ShadowHackrs/gmail-account-creator.git

# 4. 进入目录
cd gmail-account-creator

# 5. 安装依赖
pip install -r requirements.txt

# 6. 运行
python auto_gmail_creator.py
```

> ⚠️ **Termux 限制**：由于 Android 环境无法运行桌面版 Chrome，Selenium 自动化功能在 Termux 下可能受限。建议优先使用 Windows 或 Linux 桌面环境。

## 五、配置文件初始化

安装完成后，**必须**配置以下文件才能正常运行：

### 5.1 密码配置（必须）

编辑 [config/password.txt](../config/password.txt)：

```
YourStrongPassword123!
```

**密码规则：**
- 至少 8 个字符
- 必须包含：大写字母 + 小写字母 + 数字 + 特殊字符
- 必须符合 Google 账户密码策略

### 5.2 姓名库配置（必须）

编辑 [data/names.txt](../data/names.txt)，每行一个姓名：

```
Ahmed Mohamed
John Smith
Sarah Johnson
...
```

格式说明：
- 每行一个完整姓名
- 格式为 "名 姓" 或仅 "名"
- 可自由添加任意数量

### 5.3 5sim API 配置（可选，手机验证时必需）

1. 访问 [5sim.net](https://5sim.net/) 注册账户并充值
2. 在个人中心生成 API 密钥
3. 将密钥写入 [config/5sim_config.txt](../config/5sim_config.txt)：
   ```
   your_actual_api_key_here
   ```
4. 在 [config/config.py](../config/config.py) 中设置国家：
   ```python
   FIVESIM_COUNTRY = "usa"      # 可选：usa, russia, ukraine 等
   FIVESIM_OPERATOR = "any"     # 可选：any, virtual 等
   ```

### 5.4 通用配置（必须）

编辑 [config/config.py](../config/config.py)：

```python
# 账户配置
YOUR_BIRTHDAY = "2 4 1950"    # 格式："月 日 年"，必须年满 18 岁
YOUR_GENDER = "1"             # 1=男, 2=女, 3=其他
YOUR_PASSWORD = ""            # 留空则从 password.txt 读取

# 5sim 配置
FIVESIM_API_KEY = ""          # 留空则从 5sim_config.txt 读取
FIVESIM_COUNTRY = "usa"
FIVESIM_OPERATOR = "any"

# 姓名配置
USE_ARABIC_NAMES = True       # True 使用阿拉伯姓名风格
NAMES_FILE = "data/names.txt" # 姓名文件路径

# User Agent 配置
USER_AGENTS_FILE = "config/user_agents.txt"
```
